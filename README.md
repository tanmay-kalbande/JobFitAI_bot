# JobFit

AI-powered resume builder that helps you create professional, ATS-optimized resumes tailored to specific job descriptions.

## What It Does

- **Generate Resumes** — Paste your career details and get a professionally structured resume
- **Tailor for Jobs** — Add a job description and the AI will customize your resume to match
- **ATS Optimization** — Extract keywords and optionally embed them for applicant tracking systems
- **Smart Analysis** — Get a score, strengths, weaknesses, and improvement suggestions
- **Version History** — Save multiple versions for different job applications
- **One-Click PDF** — Export with smart naming (your name + company name)

## AI Providers

Works with your choice of AI:
- Google AI (Gemini)
- Cerebras
- Mistral AI

Just add your API key in Settings.

## Getting Started

1. Clone this repository
2. Run `npm install`
3. Run `npm run dev`
4. Open the app and add your AI API key in Settings

## Features

| Feature | Description |
|---------|-------------|
| Resume Generation | Create structured resumes from raw text input |
| Job Tailoring | Automatically adjust resume content for specific roles |
| ATS Keywords | Extract and optionally hide keywords for ATS scanning |
| Resume Analysis | AI-powered scoring with actionable feedback |
| Version Control | Keep multiple resume versions organized |
| PDF Export | Download with professional naming format |
